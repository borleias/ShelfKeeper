feat(admin): Implement remaining admin user management features

This commit completes the administrator access for global user management by implementing the remaining features:

- **Update user data:** Admins can now update the email and name of any user.
- **Admin password reset:** Admins can now reset the password of any user.
- **Updated `Models.cs`** with new command objects (`UpdateUserCommand`, `AdminResetPasswordCommand`).
- **Updated `IAdminUserService` and `AdminUserService`** with corresponding methods and their implementations.
- **Updated `AdminController`** with new API endpoints for these functionalities.
- **Added/updated unit tests** in `AdminUserServiceTests` to cover the new admin methods.
- **Added/updated integration tests** in `AdminIntegrationTests` to verify the new admin API endpoints.
